<h2>How to Run: </h2>

- Clone this project
- Navigate to the project directory

<h3>Terminal></h3>

```Set-ExecutionPolicy RemoteSigned -Scope CurrentUser```

``` npm install -g yarn  ```

``` yarn  install```

``` yarn  run dev```

```Set-ExecutionPolicy Restricted -Scope CurrentUser```
